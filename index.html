HTML Tutorial / Підручник HTML

What is HTML? / Що таке HTML?

HTML is the standard markup language for creating Web pages. /
HTML є стандартною мовою розмітки для створення веб-сторінок.

  • HTML stands for Hyper Text Markup Language /
    HTML означає Hyper Text Markup Language (мова гіпертекстової розмітки)

  • HTML is the standard markup language for creating Web pages /
    HTML описує структуру веб-сторінок за допомогою розмітки

  • HTML describes the structure of a Web page /
    HTML-елементи є будівельними блоками сторінок HTML

  • HTML consists of a series of elements /
    Елементи HTML представлені тегами

  • HTML elements tell the browser how to display the content /
    Браузери не відображають HTML теги, але використовують їх для відображення змісту web-сторінки

  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc. /
    Теги HTML позначають фрагменти змісту, такі як header (заголовок), paragraph (параграф), table (таблиця) та ін.

HTML Basic Examples / HTML Основні приклади

HTML Documents / HTML Документи

All HTML documents must start with a document type declaration: <!DOCTYPE html>. / Всі HTML документи мають починатися з оголошення типу документа: <!DOCTYPE html>.
The HTML document itself begins with <html> and ends with <html>. / Сам HTML документ починається тегом <html> и закінчується тегом <html>.
The visible part of the HTML document is between <body> and <body>. / Видима частина HTML документа знаходиться поміж <body> та <body>.

Example / Приклад

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading / Мій перший заголовок</h1>
<p>My first paragraph. / Мій перший параграф.</p>

</body>
</html>

HTML Headings / HTML Заголовки

HTML headings are defined with the <h1> to <h6> tags. / HTML заголовки визначаються за допомогою тегів від <h1> до <h6>.
<h1> defines the most important heading. <h6> defines the least important heading: /
Тегом <h1> визначаються найбільш важливі заголовки. Тегом <h6> визначаються найменш важливі заголовки:

Example / Приклад

<h1>This is heading 1 / Це заголовок 1</h1>
<h2>This is heading 2 / Це заголовок 2</h2>
<h3>This is heading 3 / Це заголовок 3</h3>

HTML Paragraphs / HTML Параграфи

HTML paragraphs are defined with the <p> tag: / HTML параграфи визначаються за допомогою тега <p>:

Example / Приклад

<p>This is a paragraph. / Це параграф.</p>
<p>This is another paragraph. / Це інший параграф.</p>

HTML Links / HTML Посилання

HTML links are defined with the <a> tag: / HTML посилання визначаються за допомогою тега <a>:

Example / Приклад

<a href="https://w3schoolsua.github.io/">This is a link / Це посилання</a>

The link's destination is specified in the href attribute. / Призначення посилання вказується в атрибуті href.

HTML Images / HTML Зображення

HTML images are defined with the <img> tag. / HTML зображення визначаються за допомогою тега <img>.
The source file (src), alternative text (alt), width, and height are provided as attributes: /
Початковий файл src, альтернативний текст alt, ширина width і висота height надаються в якості атрибутів:

Example / Приклад

<img src="../images/w3schools.jpg" alt="W3Schools" width="104" height="142">

HTML Кнопки

HTML кнопки визначаються за допомогою тега <button>:

Приклад

<button>Натисніть мене</button>

HTML Списки

HTML списки визначаються за допомогою тегів <ul> (невпорядкований / маркований список) або <ol> (упорядкований / нумерований список),
з подальшими тегами <li> (пункти списку):

Приклад

<h2>Невпорядкований список HTML</h2>
<ul>
  <li>Кава</li>
  <li>Чай</li>
  <li>Молоко</li>
</ul>

<h2>Впорядкований список HTML</h2>
<ol>
  <li>Кава</li>
  <li>Чай</li>
  <li>Молоко</li>
</ol>
