What is HTML? / Що таке HTML?

HTML is the standard markup language for creating Web pages. / HTML є стандартною мовою розмітки для створення веб-сторінок.

  • HTML stands for Hyper Text Markup Language / HTML означає Hyper Text Markup Language (мова гіпертекстової розмітки)
  • HTML is the standard markup language for creating Web pages / HTML описує структуру веб-сторінок за допомогою розмітки
  • HTML describes the structure of a Web page / HTML-елементи є будівельними блоками сторінок HTML
  • HTML consists of a series of elements / Елементи HTML представлені тегами
  • HTML elements tell the browser how to display the content / Браузери не відображають HTML теги, але використовують їх для відображення змісту web-сторінки
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc. / Теги HTML позначають фрагменти змісту, такі як header (заголовок), paragraph (параграф), table (таблиця) та ін.
